IRC quote bot
=============

This is a quote bot for IRC.

Movie module
============

To enable the movie module, simply add the modules under the
modules-section of your qBot.json.

```json
{
...
	"modules": [
        "movie/add",
        "movie/list",
        "movie/find",
        "movie/info",
        "movie/rate"
	]
...
}

```
